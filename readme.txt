To install, copy contents to desired web folder.
Either enter mysql connection settings in model/settings.php or access the web site and enter the connection settings.
To create components use site/example.php as a template, create a folder inside components and place the file in there.
Once a component is uploaded, go to user and add it to your user account and refresh the page for viewing.
To create pages go to page and create a page of the desired type, edit the page and the specific page type options will show up in another tab.
Once a page is created, go to your user account and add it to your user and refresh the page for viewing.

Goto http://www.mycorecms.com for support.
